# Int
**Fast big int for C++**

Int is a compact and fast big int implementation for non-negative integers in C++. It is fast because it uses array in it's implementation.

## Features

+ \+ operator
+ \- operator
+ \* operator
+ initialization with string
+ ostream support
+ reverse
+ / operator *(soon)*
+ % operator *(soon)*
+ negative integers support *(soon)*
